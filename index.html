🧠 OKNの神経科学的背景

1. OKNの二重神経経路

① 下位反射経路（脳幹反射）

網膜 → 外側膝状体(LGN) → 視蓋前野(NOT)
→ 内側前庭核(MVN) → 小脳片葉(flocculus)
→ 外転神経核 → 眼外筋運動(slow phase)

② 皮質経路（上位制御）

網膜 → LGN → V1 → V2/V3 → MT/MST野
→ NOT/小脳片葉/前庭核 → 眼球運動出力

2. 中心固定点凝視の神経生理学的効果

  1. 脳幹OKN発火の抑制：前庭眼運動系トーンの低下
  2. 前頭眼野(FEF)の活動増強：抑制的注意制御の強化
  3. 小脳前庭ループの再チューニング：バランス安定化ゲインのリセット
  4. 視覚的重力参照の再構成：内的垂直参照(SVV)の再構築

3. 臨床的応用

  • 前庭過敏（視覚動揺型めまい）の抑制
  • 姿勢制御リセット
  • 内的垂直参照（SVV/SVH）の安定化
  • トップダウン制御能力の強化
